# Express实践

## Express + Mongo 模块

  ### 实现功能
    1. index.js组织模型、路由、中间件
    2. routes实现HTTP动词
    3. model实现数据库链接模型的schema/model
    4. middleware中间件实现request/response包装处理

  ### Mongo服务器执行
  ```
  show dbs;
  use template;
  show collections;
  use templates;
  ```

  ### Node客户端执行
  ```yarn run dev```

## Mocha 单测
  ### 执行单测

  ### 覆盖率报告
